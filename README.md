# OpenCV Image Processing

## Overview
This project demonstrates basic image processing techniques using OpenCV and Python. It includes loading and displaying images, converting images to grayscale, and detecting specific colors.

## Features
- Display images using OpenCV
- Convert images from BGR to Grayscale
- Detect and highlight specific colors
- Work with multiple sample images

## Technologies Used
- Python
- OpenCV
- Matplotlib

## Project Files
- `openCV.ipynb` – Main Jupyter Notebook
- `Dog.jpeg` – Sample image
- `Persons-Faces.jpeg` – Sample image
- `haarcascade_frontalface_default.xml` – Haar Cascade classifier

## How to Run
1. Install Python and OpenCV.
2. Open the notebook using Jupyter Notebook or Visual Studio Code.
3. Run all cells in order.
4. View the processed image results.
